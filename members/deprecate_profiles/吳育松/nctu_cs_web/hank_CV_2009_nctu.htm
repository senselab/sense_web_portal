Professional Experiences

 

8/09 -

Assistant Professor, Department of Computer Science, National Chiao Tung University

1/03 – 5/09

Research Assistant, Dependable Computing Systems Lab, Purdue University (Advisor: Saurabh Bagchi)

n Development of CIDS (Collaborative Intrusion Detection System), an IDS framework, which employs novel alert correlation techniques to decrease false alarm and missed alarm rates.

n Development of Adepts intrusion response system. Adepts provides automatic online responses to deal with multi-stage attacks. We investigate on the optimality of the chosen responses. We also come up with adaptation techniques to deal with attack variants and potentially inaccurate initial Adepts parameter settings.

n Data dissemination protocol for sensor network. We develop the SPMS protocol, in which a shortest path heuristic is incorporated to reduce the energy cost of the previous SPIN protocol.

n Architecture support for robust software execution. We utilize excessive computing power to perform runtime check on running programs. A prototype based on SimpleScalar simulator for performing runtime memory leakage check is built.
